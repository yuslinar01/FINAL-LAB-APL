Keterangan Tugas :
Pattern : Strategy Pattern dan Adapter Pattern
Studi Kasus : Year
Keterangan : 
	Pada studi kasus saya membuat sebuah program dimana program tersebut akan meminta user untuk memilih salah satu hal yang ingin user ketahui, yang mana terdapat 2 pilihan yaitu year dan year Adapter. Apabila user memilih year maka user kembali diminta untuk memilih salah satu lagi. Setelah dipilih maka program akan menampilkan penjelasan dari pilihan user. Apabila user memilih year Adapter maka akan ditampilkan abad dan triwulan yang di convert kentahun

	Penggunaan strategy pattern nya yaitu pada saat pemilihan tahun. Sedangkan penggunaan adapter pattern nya adalah pada saat mengubah abad dan triwulan menjadi 2 tahun.
